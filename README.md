🌤️ WeatherCast — Advanced Real-Time Weather Dashboard

An interactive and visually dynamic weather dashboard built using HTML, CSS, and JavaScript.
WeatherCast provides real-time weather data, detailed forecasts, and a modern animated UI experience.

🌐 Live Demo
🔗 Live Site:  https://legendary-pegasus-e0147a.netlify.app/


Features : 
- City search with live preview
- Current location detection
- Real-time temperature & “feels like”
- Humidity, pressure, wind, UV index & air quality
- 48-hour forecast with temperature trend chart
- 7-day weather forecast
- Interactive map integration
- Dynamic animated weather backgrounds (day/night modes)
- Auto-refresh weather updates
